入園について

園の特色

子どもたちの「やりたい!」を大切にする保育

自然とあそぶ

園庭の大きなケヤキの木の下で、四季を感じながら外遊び。虫探し、泥んこ遊び、水たまりジャンプ。五感をフルに使って遊びます。

音楽であそぶ

リトミック、手作り楽器、季節の歌。音楽を通じてリズム感と表現力を育みます。年に一度のミニコンサートはパパママにも大人気。

食で育つ

給食は園内調理。旬の食材を使い、アレルギーにも個別対応。畑で育てた野菜を収穫して食べる食育プログラムも。

活動プログラム

年齢に合わせた多彩なプログラムで、子どもの可能性を広げます

体操

週1回 / 3〜5歳児

専任講師による体操教室。マット運動や鉄棒、跳び箱など、楽しみながら体の使い方を学びます。

リトミック

週1回 / 全クラス

音楽に合わせて体を動かし、リズム感や表現力を養います。年齢に応じたプログラムを用意しています。

英語あそび

月2回 / 4〜5歳児

ネイティブ講師と一緒に歌やゲームを通じて、自然に英語に親しみます。

食育クッキング

月1回 / 3〜5歳児

季節の食材を使ったクッキング体験。食への関心と感謝の気持ちを育みます。

おさんぽ探検

週2〜3回 / 全クラス

近隣の公園や自然豊かな道を散歩。季節の草花や虫を観察し、発見の喜びを味わいます。

制作あそび

毎日 / 全クラス

絵の具、粘土、折り紙など様々な素材を使った制作活動。自由な発想で創造力を伸ばします。
